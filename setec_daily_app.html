O navegador prepara o lote; o script PowerShell faz renomeio, Word, PDF, captura do Assunto e fechamento do arquivo.
como funciona
1. Informe a pasta e os placeholders usados no modelo do Word.
2. Confira a pasta pela pre-visualizacao.
3. Rode o comando copiado no PowerShell.
4. O script renomeia para a sequencia, preenche numero/data, salva em PDF, copia o Assunto para CSV/clipboard e fecha o Word.
